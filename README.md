Hardware requirements
-----
- Arduino Nano OR Wemos D1 OR Raspberry Pi Zero
- 2x Servo Motor (SG90)
- 2x push buttton
- 2x LED
- 2x 10k resistor
- 2x 220k resistor
- wires


Pin and board connection
-----

### Arduino NANO      

Nano | Components
------------ | -------------
D12 | Servo Motor 1
D11 | Servo Motor 2
D8 | LED 1
D7 | LED 2
D3 | Push Button 1
D2 | Push Button 2


<p align="center">
   <img src="https://github.com/ManolescuSebastian/Droid-Status/blob/master/res/droid_status_breadboard.png" width="80%"></img>

### ESP8266 (Wemos D1 mini)

Wemos D1 mini | Components
------------ | -------------
D6 | Servo Motor 1
D5 | Servo Motor 2
D8 | LED 1
D7 | LED 2
D2 | Push Button 1
D1 | Push Button 2      

<p align="center">
   <img src="https://github.com/ManolescuSebastian/Droid-Status/blob/master/res/wemosd1_droid_status_schetch_bb.jpg" width="80%"></img>
</p>

### Raspberry pi     

Raspberry pi  | Components
------------ | -------------
D12 | Push Button 1
D13 | Push Button 2
GPIO17 | Push Button 1
GPIO27 | Push Button 2
GPIO2 | LED 1
GPIO3 | LED 2
