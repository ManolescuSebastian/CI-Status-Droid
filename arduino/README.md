# Droid arduino

Hardware requirements
-----
- Arduino Nano OR Wemos D1
- 2x Servo Motor (SG90)
- 2x push buttton
- 2x LED
- 2x 10k resistor
- 2x 220k resistor
- wires

Pin and board connection
-----

### Arduino NANO      

Nano | Components
------------ | -------------
D12 | Servo Motor 1
D11 | Servo Motor 2
D8 | LED 1
D7 | LED 2
D3 | Push Button 1
D2 | Push Button 2

### ESP8266 (Wemos D1 mini)

Nano | Components
------------ | -------------
D6 | Servo Motor 1
D5 | Servo Motor 2
D8 | LED 1
D7 | LED 2
D2 | Push Button 1
D1 | Push Button 2
